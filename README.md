# EMSC2010_IndividualProject_Volcanoes-

This project analyses global volcanic eruption patterns using time series methods, statistical modelling, and randomness testing to investigate whether eruptions show any meaningful correlation with time or occur in random patterns.

The dataset contains recorded volcanic eruptions (including “Last Known Eruption” dates) which were cleaned, filtered, and standardised before analysis. Multiple approaches were used, including linear regression, polynomial regression, moving window analysis, Monte Carlo simulation, and bootstrap testing. These methods were used to compare trend-based behaviour with randomness-based models in order to better understand whether eruption frequency changes over time or is driven by stochastic processes.

**Key figures produced:**

1. Global spatial mapping of volcano locations and types
2. Yearly and decadal eruption frequency plots
3. Trend analysis using linear, polynomial, and Bayesian models
4. Moving regression to test stability of trends over time
5. Monte Carlo simulations to test deviation from randomness
6. Bootstrap analysis to test internal consistency of eruption clustering

Overall, the results suggest that volcanic eruptions do not follow a strong long-term time-based trend, but do show short-term variability and clustering depending on the statistical framework used.
